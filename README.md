#  xiaoshan-cli
![Npm 版本](https://img.shields.io/badge/xiaoshan-cli_v0.0.1-green)

## 📖简介
- 用于快速搭建前端项目的命令行工具

## 📕 模板
-  ✅ 快速生成 [Vue3+TS+Vite+ElementPlus+Pinia+Axios+Mock的模板](https://gitee.com/sohucw/admin-pro)

## 📦 安装

```bash
npm install xiaoshan-cli -g
```
## 🚩使用

```bash
# 使用模板创建新项目
xiaoshan create 

# 执行项目文件夹名称并选择模板创建新项目
xiaoshan create [projectName]

# 更新脚手架到最新版本
xiaoshan update

# 查看脚手架版本
xiaoshan -v
xiaoshan --version

```
